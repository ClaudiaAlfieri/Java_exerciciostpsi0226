## 📌 About This Repository

This repository contains the practical exercises on **Object-Oriented Programming (OOP)** developed during the **CET TPSI** programme at **ATEC — Academia de Formação**.

Each folder corresponds to a topic or set of exercises covered in class, organized progressively as the course material advances. 🎯

---

## 🧠 Concepts Covered

- ✅ Classes, Objects and Constructors
- ✅ Encapsulation (`private`, `public`, `protected`)
- ✅ Inheritance and code reuse
- ✅ Polymorphism and method overriding
- ✅ Abstract classes and methods
- 🔄 Interfaces
- 🔄 Exception Handling
- 🔄 Collections (`ArrayList`, `HashMap`, etc.)
- 🔄 Streams and Lambdas

---

## 🛠️ Technologies

- **Language:** Java
- **IDE:** IntelliJ IDEA
- **Version control:** Git + GitHub

## ▶️ How to Run

1. Clone the repository:
```bash
git clone https://github.com/teu-usuario/java-poo-tpsi.git
```

2. Open the project in **IntelliJ IDEA**

3. Navigate to the desired exercise folder and run the `Main.java` file
